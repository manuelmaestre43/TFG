# 📊 Análisis de Churn  — Trabajo de Fin de Grado

Este repositorio recoge el código y los datos empleados en el **Trabajo de Fin de Grado (TFG)** para el análisis del **churn o abandono de clientes** en una **empresa del sector de las telecomunicaciones**.  

El objetivo del proyecto es estudiar los factores que influyen en la pérdida de clientes y reproducir el flujo completo de análisis desarrollado en el TFG.

El proyecto se estructura en un único notebook principal, acompañado de la carpeta con los datos necesarios para su correcta ejecución.

## 📁 Estructura del repositorio

```bash
.
├── DATOSTFG.zip              # Carpeta con los datos utilizados en el TFG
├── AnalisisChurn.ipynb    # Notebook principal con todo el análisis
├── README.md
```

## ▶️ Ejecución del proyecto

Para ejecutar correctamente el notebook se recomienda usar **Google Colab**, ya que el proyecto ha sido preparado para trabajar con archivos almacenados en **Google Drive**.

1. Sube a tu **Google Drive** la carpeta `DATOSTFG` junto con el notebook `AnalisisChurn.ipynb`, manteniendo ambos en el mismo directorio.
2. Abre el archivo `AnalisisChurn.ipynb` con **Google Colab**.
3. Ejecuta todas las celdas desde **Entorno de ejecución → Ejecutar todo**.
4. Acepta la conexión con Google Drive.
